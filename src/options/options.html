Translation Settings / 翻訳設定

Turn on to enable AI-powered translation from any language to Japanese.
AIによる自動翻訳を有効にします（任意の言語から日本語へ）。

Select the AI provider for translation. Gemini offers a free tier.
翻訳に使用するAIプロバイダーを選択します。GeminiはFreeプランが利用可能です。

Get your API key from Anthropic Console
APIキーはこちらから取得: Anthropic Console

Select the AI model for translation. Haiku 4.5 is recommended for most use cases.
翻訳に使用するAIモデルを選択します。Haiku 4.5が推奨です。

Limit the maximum tokens per translation section to control cost (Issue #130).
Anthropic default: 4096 / Gemini default: 8192. Lower values reduce cost but may cut off long sections.
翻訳1セクションあたりの最大出力トークン数（コスト管理用）。
Anthropicデフォルト: 4096 / Geminiデフォルト: 8192。低い値にするとコストが下がりますが、長いセクションが途中で切れる場合があります。

Get your free API key from Google AI Studio
無料APIキーはこちらから取得: Google AI Studio

All models above support the free tier. 2.0 Flash is recommended.
上記モデルはすべてFreeプランに対応。2.0 Flashが推奨です。

Saves both original and translated versions. You can switch between them in the UI.
原文と翻訳の両方を保存します。UI上で切り替えが可能です。

Use {content} as a placeholder for the text to be translated. The system will replace it with the actual Markdown content.
{content} を翻訳対象テキストのプレースホルダーとして使用します。実際のMarkdownコンテンツに置換されます。

Copy Template Settings / コピーテンプレート設定

Templates to prepend when copying Markdown from the side panel. Select a template in the side panel, then click Copy.
サイドパネルでMarkdownをコピーする際に先頭に追加するテンプレートです。サイドパネルでテンプレートを選択してからコピーを押してください。

Export Settings / エクスポート設定

Automatically translates content after extracting from webpage. Requires translation feature to be enabled.
ウェブページからの抽出後に自動的に翻訳します。翻訳機能を有効にする必要があります。

Image Settings / 画像設定

Download and store images for offline viewing.
オフライン閲覧用に画像をダウンロードして保存します。
Note: Downloaded images are for personal use only. Please respect copyright and terms of service of the original content.
注意: ダウンロードした画像は個人使用のみです。著作権およびオリジナルコンテンツの利用規約を遵守してください。
